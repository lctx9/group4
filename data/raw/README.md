# TestExplora Benchmark Raw Data Source

This directory contains raw input data for the **TestExplora Benchmark (Microsoft Research 2026)**.

## Dataset Specifications:
- **Total Tasks (N):** 1,552 unique tasks.
- **Repository Count:** 482 public Python repositories (created post-2023).
- **Source:** [Microsoft TestExplora Repository](https://github.com/microsoft/TestExplora).
- **Format:** Benchmark task definitions providing defective repository snapshots and documentation-derived intent validated via the automated **DocAgent** oracle.
