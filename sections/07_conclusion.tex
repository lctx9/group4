\section{Conclusion and Future Work}
\label{sec:conclusion}

This paper presented an empirical evaluation of interactive terminal execution feedback for repository-level test suite generation. Evaluated on $N=1,552$ tasks over $4,656$ runs, our multi-agent framework achieved a Pass@1 F2P rate of 35.05\% (vs. 16.60\% baseline) and a Pass@3 rate of 71.59\% ($p = 0.000000$, $\delta = 0.3018$). Future work will extend this framework to compiled languages such as Java and C++.
